# GBT PM Toolkit

This is a Claude Code skills toolkit for Product Managers at AMEX GBT. When working in this project:

- Use American English
- Keep skills concise (under 5KB)
- Reference the GBT 7-phase AI project lifecycle where relevant
- Test case categories: happy path, edge case, adversarial, out of scope
- Priority levels: must-pass, should-pass, nice-to-have
